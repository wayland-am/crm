Wayland CRM Setup
Подключи Google Sheets как базу данных. Данные будут доступны всей команде и сохранятся между сессиями.
1
Открой script.google.com → New project
2
Вставь содержимое файла Code.gs (из архива) вместо кода по умолчанию. Сохрани.
3
Нажми Deploy → New deployment. Тип: Web app. Execute as: Me. Who has access: Anyone. Нажми Deploy, дай разрешения.
4
Скопируй Web app URL и вставь ниже.
